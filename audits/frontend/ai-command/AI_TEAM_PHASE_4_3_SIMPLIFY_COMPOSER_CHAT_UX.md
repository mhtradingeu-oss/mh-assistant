# AI Team Phase 4.3 — Simplify Composer and Chat UX

## Summary
Simplified the AI Team command center labels and guidance to reduce user confusion.

## What changed
- Composer remains the only writing surface.
- Chat is clearly positioned as the response-reading area.
- Primary action label changed from "Ask Specialist" to "Ask AI Team".
- Secondary action labels were shortened:
  - Preview Draft -> Preview
  - Task Draft -> Task
  - Workflow Draft -> Workflow
  - Handoff Draft -> Handoff
  - Browser Voice -> Voice
  - Continue in Composer -> Write Follow-up Above
- Chat action labels were simplified:
  - Use in Composer -> Use Above
  - Send to Preview -> Preview
  - Route Draft -> Route
  - Read response -> Read

## UX Principle
- Composer = write instructions
- Chat = read specialist responses
- Preview = structured output
- Tools = specialist role actions
- Context = readiness/state
- History = saved outputs

## Safety
No backend changes.
No API changes.
No router changes.
No Customer Operations route/page changes.
No execution, publish, workflow, task, ticket, SLA, escalation, or approval behavior changed.

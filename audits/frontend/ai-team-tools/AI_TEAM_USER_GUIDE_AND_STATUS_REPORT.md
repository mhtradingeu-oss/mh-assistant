## What This Page Is

AI Command / AI Team is the operating surface for working with the MH-OS AI team.

It helps the user:

- ask the AI team for guidance
- work with one specialist
- work with the full team
- prepare content
- prepare media briefs
- prepare publishing packages
- prepare governance review packs
- prepare workflows and handoffs
- use Library files as source context
- route prepared work to destination pages

AI Command prepares and routes. It does not execute final backend actions.

## Safety Boundary

AI Command is a Projection + Experience surface.

It can:

- draft
- preview
- prepare
- review
- guide
- route context

It must not directly:

- publish
- send
- run workflows
- create approvals
- create CRM records
- mutate project data
- execute backend authority

Confirmed execution belongs to backend-owned destination workspaces.

## Main User Flow

1. Open AI Command.
2. Choose a specialist or Full Team mode.
3. Choose a tool from the Smart Dock or right-side Canonical Tools.
4. Open the Smart Tool Drawer.
5. Select output type, source, destination, language, and tone.
6. If the tool requires a source, choose one from Library.
7. Use the selected Library source as context.
8. Use in Composer to prepare a review-ready prompt.
9. Review the output.
10. Route the result to the correct workspace if needed.

## Library Source Flow

1. Open a source-required tool.
2. Click Change Source.
3. Library opens at Asset Workspace.
4. Select a file, image, document, or asset.
5. Click Use as Source in AI Command.
6. The system returns to AI Command.
7. The same drawer opens.
8. The selected source appears once in Selected Source.
9. Use in Composer includes the selected source as prompt context.

# 04 — UX Risk Matrix

## Questions
- Is the page title clear and aligned with the rest of the system?
- Is Composer visually obvious as the primary action?
- Are AI Team/Specialist choices clear without too much text?
- Are selected Library sources visible in the main surface or only drawer?
- Are there duplicate buttons or repeated explanations?
- Are Tool Dock actions visually prioritized?
- Are results/history/status messages readable?
- Does the page guide the user to one next best action?

## Rule
Audit only. No runtime code changes.

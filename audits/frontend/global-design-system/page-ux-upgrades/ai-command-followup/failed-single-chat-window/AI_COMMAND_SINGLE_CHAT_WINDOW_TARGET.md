# AI Command Single Chat Window Target

## Product target

AI Command must be one clean ChatGPT-like chat workspace on desktop and mobile.

## Required layout

1. Compact page header:
   - AI Command
   - short safety line only
   - no large project/market/language/readiness chips above the chat

2. One chat window:
   - chat messages/history
   - composer in the same visual shell
   - source/library icon
   - voice icon planned/disabled unless capability exists
   - send button keeps existing behavior
   - specialist dropdown inside chat controls
   - New / Recent / Settings as compact controls

3. Secondary full-width tabs:
   - Team
   - Tools
   - Output
   - Flow

## Remove or hide from primary area

- Project / Active AI / Mode / Market / Language / Readiness chip strip
- large Recent chats card
- large Guidance Connected card
- old operating flow cards above chat
- separate Specialist Conversation visual card
- narrow tool/output panels with empty space

## Safety

- No backend/API/router/app/data changes.
- No AI provider behavior changes.
- No planned specialist activation.
- No voice runtime activation.
- No publish/send/approve/execute/CRM/workflow/task behavior.

# Library Document Preview Checkpoint

## Status
Document preview fallback support is active and clean.

## Supported document handling
- PDF: visible document card + Open document
- CSV: visible spreadsheet card + Open document
- XLS/XLSX: visible spreadsheet card + Open document
- DOC/DOCX: visible document card + Open document
- TXT/MD: visible text document card + Open document
- Selected document can be sent to AI extraction

## Notes
- Inline PDF rendering is not implemented yet.
- Spreadsheet inline table rendering is not implemented yet.
- Current support provides clear document preview fallback, open action, and AI extraction route.
- Duplicate document preview JS/CSS was removed.
- Syntax checks passed.

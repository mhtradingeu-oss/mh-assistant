### Gap 2 — Live feed readiness needs a dedicated integration/provider audit

GA4, Search Console, paid, website, and custom analytics concepts exist.

Classification:
- Partially wired or unproven per live tenant.

- keyword difficulty/source notes
- search intent confidence
- market/language targeting
- competitor/context input
- evidence references

Classification:
- Strong foundation, needs product depth.

The system can generate recommendations and readiness notes.

- evidence
- confidence
- missing data
- expected impact

Classification:
- Partially wired.

Classification:
- Safety preserved.

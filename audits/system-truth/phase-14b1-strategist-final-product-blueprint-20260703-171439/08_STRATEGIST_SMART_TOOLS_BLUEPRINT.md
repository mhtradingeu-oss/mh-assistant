
Strategist Smart Tools Blueprint
Tool experience

Every tool must work in three ways:

Chat command
One-click tool card
Manual workspace action inside Campaign Studio
Tool model

Each tool should have:

label
short description
required input
optional input
local engine path
optional provider path
preview output
approve/edit/handoff actions
Final Strategist tool set

Tool safety

All Strategist tools are preview/handoff tools.

They do not:

publish
send
run workflow
mutate CRM
connect provider
spend budget


Strategist Safety / Authority Blueprint
Allowed

Strategist may:

analyze
plan
draft
compare
recommend
preview
prepare handoff packets
create task suggestions
identify missing context
recommend next actions
Review required

Strategist outputs should be reviewed when:

campaign contains regulated claims
campaign targets sensitive categories
budget or paid ads are involved
publishing is close
external provider use is requested
user asks for legal/compliance-sensitive strategy
Confirmation required elsewhere

Strategist cannot directly:

publish
send messages
mutate CRM
create final tickets
run workflows
approve governance
connect/disconnect integrations
spend ad budget
call paid providers without setting/approval
Ownership boundaries
Action	Owning workspace
Publishing	Publishing
Claims approval	Governance / Compliance
Workflow run	Workflows
Task creation confirmation	Task Center / Operations
CRM mutation	CRM/Sales workspace
Customer reply send	Customer Center
Media generation execution	Media Studio
Video provider execution	Media/Video workspace
Guard principle

Strategist produces the plan.

Owning workspace executes after explicit user confirmation.

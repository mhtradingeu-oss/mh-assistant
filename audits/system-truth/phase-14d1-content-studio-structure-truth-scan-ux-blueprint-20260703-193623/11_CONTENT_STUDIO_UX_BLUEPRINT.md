# Phase 14D.1 — Content Studio UX Blueprint

## Working Decision
Content Studio should remain as a separate workspace.

## Recommended External Name
Content Studio

## Recommended Internal Page Title
Content Production Studio

## Page Purpose
Turn approved strategy briefs into reviewed, channel-ready content.

## Primary Owner
Writer

## Review Roles
- Editor
- Admin

## Connected Specialists
- Strategist: source of campaign brief and offer/audience direction.
- Writer: primary content producer.
- Editor: quality, tone, clarity, and readiness reviewer.
- SEO Specialist: search title, metadata, keywords, search intent.
- Localization Specialist / Translator: market-language adaptation.
- Designer / Media Specialist: receives content needing visual/video assets.
- Publisher: receives approved content for scheduling.
- Ads Operator: receives approved ad copy variants.
- Compliance Reviewer: checks legal/sensitive claims when required.

## Recommended Page Layout
1. Command Header
   - Project
   - Source
   - Owner
   - Review
   - Mode
   - Current status

2. Incoming Brief Panel
   - Campaign objective
   - Audience
   - Offer angle
   - Product focus
   - Channels
   - Tone/language
   - Required assets
   - Blockers
   - Source page link/reference

3. Specialist Workspace Tabs
   - Writer
   - Editor
   - SEO
   - Localization
   - Handoff

4. Content Draft Builder
   - Content type
   - Channel
   - Language
   - Tone
   - Headline
   - Hook
   - Body/caption/script
   - CTA
   - Hashtags/keywords
   - Notes

5. Preview Panel
   - Instagram/social preview
   - Ad copy preview
   - Website/product copy preview
   - Email preview
   - Script preview

6. Review & Readiness Panel
   - Brief received
   - Draft created
   - Tone aligned
   - SEO ready
   - Legal/compliance notes
   - Media required
   - Publishing ready

7. Handoff / Next Destination Panel
   - Media Studio if visual assets are required.
   - Publishing if content is approved.
   - Ads Manager if ad copy is approved.
   - Campaign Studio if strategy is incomplete.
   - Workflows if repeatable execution is needed.

8. Activity / Source / Audit Panel
   - Source page
   - Source specialist
   - Current owner
   - Last update
   - Review state
   - Handoff history

## UX Rule
The page must not show every specialist at once. It should use tabs, panels, or a role selector to reduce crowding.

## Safety Rule
No auto-publish, no auto-send, no ad spend, no hidden provider execution, no workflow auto-run.

# B1 Header Reconstruction

## Goal
Replace the mixed legacy dashboard header with a clean international AI operating system shell.

## Problems detected
- Legacy topbar layers still active
- Mobile controls from previous UX generation
- Role and key indicators exposed globally
- Repeated workspace naming
- Visual density and zoomed feeling
- Multiple command paradigms mixed together

## New shell principles
- Minimal top header
- One workspace title
- One AI entry point
- One primary action
- Responsive mobile-first behavior
- Contextual controls instead of persistent admin pills

## Desktop target
[ Logo ] [ Workspace ] [ AI ] [ + ] [ User ]

## Mobile target
☰ [ Workspace ] [ AI ]

## Remove from global header
- roleSwitcher
- accessKeyBtn
- mobileRoleBadge
- commandToggleBtn
- context cards
- duplicated titles

## Move into user/system menu
- role selection
- developer mode
- access keys
- runtime diagnostics

## UX direction
AI-first operating system, not internal admin dashboard.

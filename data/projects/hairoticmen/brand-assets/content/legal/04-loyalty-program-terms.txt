HAIROTICMEN Loyalty Program Terms
(Rewards System)

--------------------------------------------------

§1 Allgemeines

Das HAIROTICMEN Loyalty Program ist ein freiwilliges Bonus- und Belohnungssystem für:

- Endkunden
- Partner (Salon / Retail)
- Stand Partner
- Affiliate Partner
- Vertriebsmitarbeiter

Das Ziel ist die Förderung von Aktivität, Verkauf und Markenbindung.

--------------------------------------------------

§2 Teilnahme

Teilnahme erfolgt durch:

- Registrierung im System
- Zustimmung zu diesen Bedingungen

Das Unternehmen kann Teilnehmer jederzeit ablehnen oder entfernen.

--------------------------------------------------

§3 Punkte (Points System)

Teilnehmer können Punkte sammeln durch:

### Für Kunden:
- Käufe im Online-Shop
- Wiederkäufe
- Aktionen und Kampagnen

### Für Stand Partner:
- Verkäufe
- Refill-Bestellungen
- Aktivität (z. B. Reports)

### Für Affiliate:
- vermittelte Verkäufe

### Für Partner:
- Bestellungen
- Promotion
- Kooperation

--------------------------------------------------

§4 Punktevergabe

Punkte werden:

- automatisch über das System berechnet
- basierend auf Aktionen und Regeln vergeben

Das Unternehmen behält sich vor:

- Punkte anzupassen
- Regeln zu ändern
- Fehler zu korrigieren

--------------------------------------------------

§5 Einlösung von Punkten

Punkte können verwendet werden für:

- Rabatte
- Produkte
- spezielle Angebote
- exklusive Vorteile

Punkte sind:

- nicht übertragbar
- nicht auszahlbar

--------------------------------------------------

§6 Gültigkeit

Punkte können:

- zeitlich begrenzt sein
- verfallen bei Inaktivität

Das Unternehmen bestimmt die Laufzeit.

--------------------------------------------------

§7 Missbrauch

Bei Missbrauch kann das Unternehmen:

- Punkte entfernen
- Accounts sperren
- Teilnahme beenden

Missbrauch umfasst:

- Manipulation
- Fake-Aktivität
- Systemausnutzung

--------------------------------------------------

§8 Systemintegration

Das Loyalty-System ist integriert mit:

- WooCommerce
- CRM
- Affiliate-System
- Stand Program
- AI System

--------------------------------------------------

§9 Automatisierung & KI

Das System nutzt AI zur:

- Analyse von Verhalten
- Berechnung von Belohnungen
- Optimierung von Kampagnen
- Personalisierung von Angeboten

AI kann automatisch:

- Punkte vergeben
- Level anpassen
- Bonusaktionen auslösen

--------------------------------------------------

§10 Level & Gamification

Teilnehmer können Levels erreichen, z. B.:

- Bronze
- Silver
- Gold
- VIP

Basierend auf:

- Aktivität
- Umsatz
- Engagement

--------------------------------------------------

§11 Änderungen

Das Unternehmen kann:

- Regeln ändern
- Punkte anpassen
- System erweitern

Teilnehmer werden informiert.

--------------------------------------------------

§12 Haftung

Keine Haftung für:

- Verlust von Punkten
- Systemfehler
- technische Probleme

--------------------------------------------------

§13 Beendigung

Das Unternehmen kann:

- Teilnahme beenden
- Punkte löschen

bei Verstoß oder Inaktivität.

--------------------------------------------------

§14 Datenschutz

Daten werden gemäß DSGVO verarbeitet.

--------------------------------------------------

### POINTS RULE

- Points have no monetary value
- No conversion to cash allowed

--------------------------------------------------

### GAMIFICATION RULE

Rewards must be:

- transparent
- fair
- system-based

--------------------------------------------------

### AI LOYALTY RULE

AI may:

- assign rewards
- optimize campaigns
- analyze behavior

AI must NOT:

- create unfair advantage
- override system rules

--------------------------------------------------

### SYSTEM NOTE

This document controls:

- Rewards logic
- Points system
- Gamification
- Loyalty automation
- AI reward decisions

All loyalty-related actions must follow this document.

## Boundary Notes

- Provider ID is not model ID, worker ID, or operation ID.
- Application action and adapter action are not provider operations by default.
- Execution implementation ownership does not grant security authorization.
- Conversation storage does not prove communication provider send support.
- A normalized response is not automatically an artifact contract.

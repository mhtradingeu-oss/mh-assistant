# Provider ID / Operation / Model Matrix

> This Phase 1A-3 inventory freezes observed provider vocabulary and records recommendations only. It does not approve provider IDs, provider operations, application actions, adapter actions, models, workers, credentials, configuration, health, routing, fallback, execution, certification, migrations, schemas, registries, or runtime changes.

## Notes

- Inventory granularity is explicit and grouped rows do not imply a single provider record.
- operation_namespace is separated from provider identity and model identity.
- credential_reference_state, credential_value_state, and credential_validation_state are independent.
- IMPLEMENTED_PATH is distinct from EXECUTION_SUCCEEDED and CERTIFIED_CURRENT_HEAD.

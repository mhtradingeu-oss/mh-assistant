# MH-OS Contract Ownership Map

## Principle

Existing runtime ownership remains authoritative until migration is approved.

## Domains

| Domain | Current Sources | Future Contract |
|---|---|---|
| Security | security modules | Authority Contract |
| Capabilities | multiple sources | Capability Contract |
| Providers | provider registries | Provider Contract |
| Artifacts | multiple writers | Artifact Contract |
| Versions | entity-specific fields | Version Contract |
| Memory | AI memory stores | Memory Contract |
| Governance | policy/gate modules | Governance Contract |

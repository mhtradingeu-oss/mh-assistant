# MH Assistant — Master System Overview

## 1. System Purpose

MH Assistant is a unified AI-powered Marketing Operating System designed to:

- Ingest project data and knowledge
- Understand brands, products, and markets
- Generate and manage marketing content
- Execute campaigns across channels
- Monitor, analyze, and improve performance
- Continuously learn and optimize outcomes

This system transforms any project into a scalable, data-driven growth machine.

---

## 2. Core Architecture

The system is built on a modular layered architecture:

### Layer 1 — Data & Source of Truth
- Brand assets
- Product data
- Content library
- Legal & commercial documents
- Website & platform integrations

### Layer 2 — Intelligence Engine
- Brand Intelligence
- Product Intelligence
- Audience Intelligence
- Channel Intelligence
- Campaign Intelligence

### Layer 3 — Content & Creative Engine
- Prompt Packs
- Channel Packs
- Creative Maps
- Media Prompts (image/video/audio)

### Layer 4 — Campaign & Execution Engine
- Campaign Packages
- Launch Waves
- Scheduler
- Execution Modes (semi_auto / full_auto)
- Channel Payloads

### Layer 5 — Monitoring & Optimization
- Post Review
- Content Improvement
- Performance Tracking
- Learning (Product + Channel)

---

## 3. Key System Capabilities

The system enables:

- Multi-channel marketing automation
- Intelligent content generation
- Campaign planning and execution
- Data-driven optimization
- Continuous learning and improvement
- Scalable growth operations

---

## 4. Execution Modes

### Semi-Auto Mode
- System prepares content and execution packages
- Operator reviews and publishes manually

### Full-Auto Mode (Future Layer)
- System publishes and sends automatically
- Requires platform adapters and validation layers

---

## 5. Source of Truth Priority

The system follows a strict data hierarchy:

1. Uploaded project assets (highest priority)
2. Website / eCommerce platform (WooCommerce, Shopify)
3. Connected channels (social, email, marketplaces)
4. Performance analytics (learning layer)

---

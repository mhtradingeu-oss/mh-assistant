# Customer Operations Runtime

Authoritative runtime domain for:

- customer conversations
- tickets
- unified inbox
- channel connectors
- SLA / escalation
- customer profiles
- outreach
- voice / IVR
- AI customer operations

This runtime extends MH-OS operational backbone.

Rules:
- Backend owns authority.
- Frontend projects state only.
- Reuse governance/escalation systems.
- No duplication with existing ops backbone.
- No direct production integrations before contracts stabilize.

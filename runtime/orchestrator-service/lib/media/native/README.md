# Native Media Runtime Foundation

This folder contains the safe foundation for MH-OS native media generation.

## Current scope

- Image runtime skeleton
- Video runtime skeleton
- Audio runtime skeleton
- Voice chat runtime skeleton
- Provider bridge
- In-memory queue foundation
- Output storage helper
- Native Media Intelligence Layer
- Media knowledge files

## Important

This layer does not yet perform heavy rendering.

It prepares, scores, routes, and structures media generation jobs before actual rendering engines are integrated.

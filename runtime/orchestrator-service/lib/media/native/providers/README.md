# Native Media Provider Catalog

Tracks API providers and native/open-source model targets.

Current scope:
- OpenAI image target
- Google Nano Banana / Gemini image target
- Google video target
- Runway/Kling/Pika video targets
- Native open-source image/video/audio targets
- Provider readiness by environment variables

This layer does not call provider APIs yet.

Next:
Create provider adapters that transform MH-OS worker jobs into provider-specific API calls.

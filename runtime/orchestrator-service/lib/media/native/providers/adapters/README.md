# Native Provider Execution Adapters

Transforms MH-OS native media jobs into provider-specific API executions.

Current adapters:
- OpenAI image generation adapter
- OpenAI realtime voice placeholder

Important:
- The demo only checks configuration/readiness.
- It does not call paid generation APIs automatically.

Future adapters:
- Google Gemini / Nano Banana
- Google Veo
- Runway
- Kling
- Pika
- Native GPU worker execution adapter

# Native Media Rendering Layer

This folder defines rendering adapters for native media generation.

Current status:
- Safe adapter foundation only
- No GPU execution yet
- No external provider execution
- No production behavior change

Supported future rendering targets:
- Image: FLUX, Stable Diffusion, ComfyUI
- Video: Wan, CogVideoX, LTX Video, Stable Video Diffusion
- Audio: XTTS, Coqui, Piper, OpenVoice
- Voice Chat: STT + AI agent + TTS + realtime session runtime
